# Axel Coaching Classes, Sahdoura

Axel Coaching Classes is a dedicated educational platform based in Sahdoura, Datia, Madhya Pradesh. Our mission is to provide quality education and vocational training to students from class 1st to 10th.

## Features
- **Comprehensive Learning**: Structured course details for students.
- **Resource Library**: Access to essential notes and study materials.
- **Live Updates**: Stay connected with upcoming classes and important announcements.
- **Student Dashboard**: A personalized interface to track progress and study schedules.

## Technologies Used
- HTML5, CSS3, JavaScript
- Firebase (for authentication and database)
- Hosted on Netlify/GitHub Pages

## About
Founded by Vishnuyash Thakur to empower local students through digital education.
